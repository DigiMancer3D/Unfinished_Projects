soloPAW


        We are gamers that play 1-handed. We do speedruns, percent-runs, quick plays, any runs & playthroughs of video, flash, board, card & social games. Not all gamers are fully able-bodied, fully mentally capable, owning the new systems & can afford full version/new games. We are the odd, unusal & different when it comes to gaming.

We are Gammers of a different breed!


        The tools the gammer uses can often times be the difference during a learning phase of a game or can give you a cutting edge to the hardest competitions! What tool YOU use is up to You but the main tool used by many games old or new, the computer mouse. soloPAW Players use gaming mice like the G502 by Logitech which is where our main-state comes from, Disfigured Tiger holding a Gaming Mouse (in landing picture), & our namestate, soloPAW, holding a computer mouse.

        This page is to not only bring in awareness about disabled, new, different & unusal gammers but also to give them a place to collect their runs, run types and more to help bring in sPAW run categories. Starting with the sPAW categories, keep in mind at the footer of this page is a way to submit your sPAW runs, categories & proofs to add to this page, they are as follows:
    A) Demo Runs [demo filename/disc]: Play through a demo disc/game in the various run types. **intro/fun/rare speedrun**

    B) Playthroughs: Show the road to beating a game. May be technical, vauge or just tips. **informative speedrun**

    C) Any Runs [run name/location]: Play through a section of a game or do specific areas/bosses/puzzles/challenges runs for a game. **informative speedrun**

    D) Quick Plays [section/run]: Play through a section of a game or do specific areas/bosses/puzzles/challenges runs for a game as fast as possible. **technical speedrun**

    E) Percent-Runsn [--%]: Play through a game and complete with a fully completed percentage (only up to 99%) as fast as you can. **technical speedrun**

    F) Speedrun: Play through a game and complete with a fully completed percentage (100%) as fast as you can. **technical speedrun**

    G) Speedrun: Play through a game and complete with a fully completed percentage (100%) as fast as you can (without exiting the game, timer never pauses). **original speedrun**

        Run types can be various of almost any kind. These are community created and when appropriate is seen inside brackets ( [ ] ) next to the speed run filename/name when uploaded. Most of the way you upload your videos can help your videos be found through the sPAW and soloPAW searches.
        Naming your video to reflect your run on public forums (like YouTube, Odysee, Rumble, etc) where anyone without an account can view the video can make your run easier to find. Set the title in similar to the following example, "Game Title sPAW Character next to speedrun [run type] Run (end-time)". This will tell the majority of information and is the same data requested when uploading proofs. This is the default way to name sPAW runs. If your run doesn't have a time, type or another section of the titling, just don't include that section. For example if your run has no type it may look like, "Elden Ring sPAW G Run (1:33:01)", and this example would be a legit sPAW run that needs no run type (because it's 100% need to beat plus dedicated restrictions).

        In the page footer is the way to uploading your runs and it's a simple text comment in a Shoutbox. Set up your comment in the same way you set up your video title. This simple and decentralized way to submit runs does require your comment, name & info to match your video uploaded. To keep a single profile, re-use the same public forum account and Shoutbox data for other runs.Runs may have their own run uploading Shoutbox so only submit new run types or runs that would require new sections to be created. Anyone can crete a section or page for this website and there are Shoutboxes for submitting those upgrades as well.
        Sumbitting site upgrages can be done in similar fashion as uploading your run videos, "Tab Title What's being upgraded [keyword(s) for this upgrade] Where the code is hosted (code link)". Tab title is the title that will be in the tab (must be short). What's being upgraded may be like new page, new navbar, new page effects etc etc etc. Keyword may be cosmetics or new feature but generally should explain the point of the upgrade in 1 to 3 words. Where the code is hosted should match link data and where the following link directs to. The code link is the link to the code for the upgrade. This site is hosted on IPFS/IPNS and is looking to one day have IPNS folders for each sPAW Run so uploading would be submitting through a decentralized uploader or self-uploading to the folder and it would appear on the site automatically. Submitting upgrades for this upgrade first could be the diffrence between anyone submiting runs to only more technical capable people submitting. Our goal for submitting is to have a decentralized and easy way to upload.
